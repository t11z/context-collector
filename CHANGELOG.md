## [0.1.0] - 2026-04-07

### Added
- Initial release
- Topic-based file collection via `.context-collector.toml`
- Free-form path selection with `--paths`
- Dry-run mode
- Size warnings for large outputs
- Stdout output for piping
- Default exclusions for common generated files and binaries
- Language detection for 30+ file extensions
- Table of contents in output
- CI workflow for tests and linting
